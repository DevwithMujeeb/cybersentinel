# 🛡️ CyberSentinel

### Autonomous Network Threat Detection Agent

CyberSentinel is an AI-powered autonomous agent that monitors your network, detects threats, and logs recommendations — all without human intervention.

## 🚀 Features

- 🔍 Autonomous network port scanning
- 🤖 AI-powered threat analysis (GitHub Models / GPT-4o-mini)
- 📊 Live threat dashboard
- 📋 Real-time threat logging with timestamps
- 🔄 Runs continuously every 60 seconds

## 🛠️ Tech Stack

- Node.js + Express
- GitHub Models API (GPT-4o-mini)
- HTML/CSS/JavaScript Dashboard

## ⚙️ Setup

1. Clone the repo

```bash
   git clone https://github.com/DevwithMujeeb/cybersentinel.git
   cd cybersentinel
```

2. Install dependencies

```bash
   npm install
```

3. Create `.env` file
4. Run the agent

```bash
   node server.js
```

5. Open browser at `http://localhost:3000`
